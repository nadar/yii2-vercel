# Flyo Nitro Yii2 Example

A Flyo, Yii2, Vite, Tailwind and Bref (Servless) Kickstarter Project.

## Running

vite & tailwind:

1. npm i
2. npm run dev

server:

1. `composer install`
2. `composer serve`
3. visit `localhost:8888`
